# Network Subnet Tool

Простой CLI-инструмент на Go для работы с IP и подсетями.

## Функции
- Парсинг IP и маски
- Вычисление сетевого и broadcast адреса
- Определение диапазона хостов
- Генерация всех подсетей в сети

## Пример использования
```bash
go run cmd/main.go
Enter IP address: 172.16.5.10
Enter subnet mask: 255.255.255.224
